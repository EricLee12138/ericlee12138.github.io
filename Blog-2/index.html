[Strings] Reverse Integer

Given a 32-bit signed integer, reverse digits of an integer.

Example 1

Input:  123
Output: 321

Example 2

Input:  -123
Output: -321

Example 3

Input:  120
Output: 21

Note

Assume we are dealing with an environment which could only store integers within the 32-bit signed integer range: \([−2^{31}, 2^{31} − 1]\). For the purpose of this problem, assume that your function returns 0 when the reversed integer overflows.

InitialThoughts

  1. Get each digit number of the input number and store them into a vector.
  2. Use two iterators to iterate from the beginning and the end and swap two values of the iterators, therefore, get the reversed number.
  3. Check the validity of the number by comparing reversed digits with the digits of the biggest integer INT_MAX.
  4. Combined valid digits into one integer and return, otherwise, return 0.
int reverse(int x) {
    vector<int> bits, max_bits;
    int a, b;

    int i = 1;
    while (i < 1000000000) {
        a = x / i;
        b = x / (i * 10);

        if (a == 0 && b == 0)
            break;

        bits.push_back(a - b * 10);
        i *= 10;
    }
    if (x / i != 0)
        bits.push_back(x / i);

    int j = 1;
    while (j < 1000000000) {
        a = INT_MAX / j;
        b = INT_MAX / (j * 10);

        if (a == 0 && b == 0)
            break;

        max_bits.push_back(a - b * 10);
        j *= 10;
    }
    max_bits.push_back(INT_MAX / j);

    if (bits.size() <= 0) {
        return 0;
    }

    vector<int>::iterator begin = bits.begin();
    vector<int>::iterator end = bits.end() - 1;

    while (begin < end) {
        int temp = *begin;
        *begin = *end;
        *end = temp;

        begin++; end--;
    }

    if (bits.size() == max_bits.size()) {
        for (int j = bits.size() - 1; j >= 0; j--) {
            if ((x > 0 && bits[j] > max_bits[j]) ||
                (x < 0 && -bits[j] > max_bits[j]) ) {
                return 0;
            } else if ((x > 0 && bits[j] < max_bits[j]) ||
                       (x < 0 && -bits[j] < max_bits[j])) {
                break;
            }
        }
    }

    int y = 0;
    for (int j = 1, k = 0; k < bits.size(); j *= 10, k++) {
        y += j * bits[k];
        if (j == 1000000000)
            break;
    }
  
    return y;
}

Reference Solution

int reverse(int x) {
    int rev = 0;
    while (x != 0) {
        int pop = x % 10;
        x /= 10;
        
        if (rev > INT_MAX / 10 ||
            (rev == INT_MAX / 10 && pop > 7))
            return 0;
        
        if (rev < INT_MIN / 10 || 
            (rev == INT_MIN / 10 && pop < -8))
            return 0;
        
        rev = rev * 10 + pop;
    }
    
    return rev;
}

[Linked List] Remove Nth Node From End of List

Given a linked list, remove the n-th node from the end of list and return its head.

Example

Given linked list: 1->2->3->4->5, and n = 2.
After removing the second node from the end, the linked list becomes 1->2->3->5.

Note

Given n will always be valid.

Follow up

Do this in one pass.

Initial Thoughts

Accomplishing the task in two passes is intuitive, by calculating the size of the linked list first and then find the nth node from the end.

Using two pointers on the linked list, we can do this in one pass. The main idea is that, the first pointer points to the head node, the second one points to the nth child of the head node, between which there is a n-1 gap. Then move the two pointers at the same time until the second pointer reaches the end, the first pointer then points to the very node to be deleted.

ListNode* removeNthFromEnd(ListNode* head, int n) {
    ListNode*& first = head;
    ListNode* second = first;
    for (int i = 0; i < n; i++) {
        second = second->next;
    }

    while (second != NULL) {
        first = first->next;
        second = second->next;
    }

    // TODO: delete the node where the first pointer points

    return head;
}

What is difficult is how to correctly delete the node. The problem is that the node might be the tail node of the linked list. Directly deleting it as what is done in Delete Node in a Linked List may cause runtime error since it doesn’t handle the tail node correctly.

The right solution should be to find the parent of the first node instead of the first node. One way to do this is to extend the original linked list by adding a node to its head and then maintain a constant n gap.

ListNode* removeNthFromEnd(ListNode* head, int n) {
    ListNode* parent = new ListNode(0);
    parent->next = head;

    ListNode* first = parent;
    ListNode* second = parent;

    for (int i = 0; i <= n; i++) {
        second = second->next;
    }

    while (second != NULL) {
        first = first->next;
        second = second->next;
    }

    first->next = first->next->next;

    return parent->next;
}

Example

Input:  1->2->3->4->5->NULL
Output: 5->4->3->2->1->NULL

Iterative Solution

ListNode* reverseList(ListNode* head) {
    if (head == NULL)
        return head;

    ListNode *current = head->next;
    ListNode *last = head;
    ListNode *temp;

    head->next = NULL;

    while (current != NULL) {
        temp = current->next;
        current->next = last;
        last = current;            
        current = temp;
    }

    return last;
}

[Linked List] Merge Two Sorted Lists

Merge two sorted linked lists and return it as a new list. The new list should be made by splicing together the nodes of the first two lists.

Example

Input:  1->2->4, 1->3->4
Output: 1->1->2->3->4->4

Solution

ListNode* mergeTwoLists(ListNode* l1, ListNode* l2) {
        if (l1 == NULL) {
            return l2;
        }
        
        if (l2 == NULL) {
            return l1;
        }
        
        ListNode* l1First = l1;
        ListNode* l1Second = l1->next;
        ListNode* l2Target = l2;
        
        if (l2Target->val <= l1First->val) {
            l2Target->next = mergeTwoLists(l1First, l2Target->next);
            return l2Target;
        }
        
        if (l1Second == NULL) {
            l1First->next = l2Target;
            return l1First;
        }
        
        if (l2Target->val > l1First->val && l2Target->val <= l1Second->val) {
            l2Target->next = mergeTwoLists(l1Second, l2Target->next);
            l1First->next = l2Target;
            return l1First;
        }
        
        if (l2Target->val > l1Second->val) {
            l1Second->next = mergeTwoLists(l1Second->next, l2Target);
            return l1First;
        }
        
        return NULL;
    }

[Linked List] Find The Middle

Given a singly linked list, find middle of the linked list. For example, if given linked list is

1->2->3->4->5

then output should be 3.

If there are even nodes, then there would be two middle nodes, we need to print second middle element. For example, if given linked list is

1->2->3->4->5->6

then output should be 4.

Initial Thought

Intuitively, the middle node should be the ((int)size / 2) + 1 node of the linked list, where size is the overall size of the linked list. So we can find it by counting.

int FindMiddle(ListNode* head) {

    // Get the overall size of the list
    ListNode* cur = head;
    int size = 0;
    while (cur != NULL) {
        size++; cur = cur->next;
    }

    // Return the ((size / 2) + 1)th node of the list
    cur = head;
    int middle = size / 2;
    while (middle > 0) {
        middle--; cur = cur->next;
    }

    return cur->val;
}

The method uses two integers, one linked list node and traverses the list for one and a half times.

Two Pointers Method

Traverse linked list using two pointers. Move one pointer by one and other pointer by two. When the fast pointer reaches end slow pointer will reach middle of the linked list.

Oddly Increment Method
