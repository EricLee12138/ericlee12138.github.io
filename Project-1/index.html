Description

Hexahedron is a game world that can be transformed between 2D plane and 3D space, inspired by Fez. The game world can be really impressive because a simple change of perspective can make the whole world different.

Players will help the character, Hexa, by repeatedly changing the perspective to collect coins and stars, defeat and avoid different enemies and finally reach the goal.

Demo

The game is built basically out of Three.js, which can easily run on most latest web browser, e.g. Safari, Chrome, etc., through this link. Noted that quite a few models need to be imported into the canvas, it might take 2-3 minutes first time loading the game.
