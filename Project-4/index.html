Card

Web

Overview

Duration: 2 weeks

Role: Programmer

Team: Solo

Platform: Mobile Web Browser

Description

Card works as a to-do list with following basic functions:

  • Add, delete, complete, cancel, and delete individual items.
  • Complete and cancel all items, and clear completed items.
  • Save the list using LocalStorage, restore the page (and even close the browser) and still recover.

Besides, Card has following features and styles:

  • With the concept of folder, allow users to save multiple lists, and provide simple basic operations on those folders.
  • Edit the contents of a folder or item again after creation.
  • Filter completed or unfinished items
  • Provide basic searching function, filter items with keywords.
  • Apply iOS12 stacked notification center, which is easy to interact with.
